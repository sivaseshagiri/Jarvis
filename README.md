JARVIS - Personal AI Assistant

A sci-fi inspired personal AI assistant built with Python and HTML.
 Features
- 🎙️ Voice input and output
- 🤖 LLM-powered responses (Groq API)
- 💬 WhatsApp Web automation via Chrome Extension
- 🚀 App launching and system controls
- 🔊 Volume control
- 📸 Screenshots
- 📋 Clipboard access
- ⏰ Reminders
- 🌤️ Weather info
- 📁 File browsing

Tech Stack
- Python (backend via WebSocket)
- HTML/CSS/JavaScript (frontend)
- Chrome Extension (Manifest V3)
- Groq LLM API
Project Structure
- `bridge.py` - Python backend server
- `JARVIS.html` - Sci-fi animated frontend
- `manifest.json` - Chrome extension config
- `content.js` - WhatsApp Web automation
